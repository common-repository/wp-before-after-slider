Before After Image slider

== Description ==

WP Before After Slider makes it easier for you to show your clients the differences between the original and the new image. Save your time using this quick and easy-to-use plugin.

A powerful and useful plugin for Bloggers, Graphic Designers, Beauticians, Doctors, Photographers, News Websites.

=== Features ===
- Easy to use
- No Coding Skills Required
- Easy to use Admin Panel
- Responsive & Mobile Friendly
- Light-Weight & Compatibility
- Lots of Customization Options
- Quick Support
- Many more to come...

** Easy to use **
Create new Slider, Upload Before and After slide images, Configure slider options, Done

** No Coding Skills Required **
After selecting the slider it will generate simple shortcode, copy and paste it wherever you want to show the slider.

** Easy to use Admin Panel **
Simple, user-friendly Admin Panel settings make it easier for you to make changes quickly. 

** Mobile Friendly & Responsive **
Mobile / Touch friendly and responsive to showcase your work on any screens.

** Light-Weight & Compatibility **
The WP Before After slider is built Lightweight and flexible to work with any Page Builder or Design.

** Lots of Customization Options **
You can easily customize text, color, slider design options.

**Quick Support**
Our dedicated support team is always there to provide you with quick and full support for any issues related to this plugin.


== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "WP Before After Slider".
3. Click install.
4. Click activate.
